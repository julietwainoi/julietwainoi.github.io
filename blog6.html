Terraform keeps track of your infrastructure with a file called terraform.tfstate. To manage this state securely and support team collaboration, it's best to store it remotely using an S3 backend.

Here is an example configuration to store Terraform state in an S3 bucket with state locking via DynamoDB:

terraform {
  backend "s3" {
    bucket         = "my-terraform-state-bucket"
    key            = "envs/dev/terraform.tfstate"
    region         = "us-west-2"
    dynamodb_table = "terraform-lock-table"
    encrypt        = true
  }
}
