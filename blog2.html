Date: May 27, 2025

Author: Juliet Wanjiku (@julietwainoi)

📘 What I Learned

  • How to define infrastructure in code using Terraform
  • How to launch an EC2 instance using a main.tf configuration file
  • The structure of a Terraform project and how to use terraform init, plan, and apply
  • How to set up a simple web server using user_data
  • Security groups and basic access control

🛠️ Infrastructure Details

Single Server Deployment

  • Region: us-east-1
  • Instance Type: t2.micro
  • Security Group: Allows SSH (port 22)

Web Server Deployment

  • Region: us-east-1
  • Instance Type: t2.micro
  • Security Group: Allows HTTP (port 80)
  • Extras: Used user_data to install and start NGINX on boot

📌 Sample Terraform Configuration


resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
  user_data     = <<-EOF
              #!/bin/bash
              sudo apt update
              sudo apt install -y nginx
              systemctl start nginx
              systemctl enable nginx
            EOF

  tags = {
    Name = "WebServer"
  }
}
