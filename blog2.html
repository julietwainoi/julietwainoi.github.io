📘 What I Learned

  • Defining infrastructure using Terraform
  • Launching an EC2 instance with a main.tf file

  • Configuring a web server with user_data
  • Understanding basic AWS security groups

🛠️ Infrastructure Details

Single Server Deployment

  • Region: us-east-1
  • Instance Type: t2.micro
  • Security Group: Allows SSH (port 22)

Web Server Deployment

  • Region: us-east-1
  • Instance Type: t2.micro
  • Security Group: Allows HTTP (port 80)
  • Extras: Installs NGINX via user_data

📌 Sample Code

resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
  user_data     = <<-EOF
    #!/bin/bash
    sudo apt update
    sudo apt install -y nginx
    systemctl start nginx
    systemctl enable nginx
  EOF

  tags = {
    Name = "WebServer"
  }
}
